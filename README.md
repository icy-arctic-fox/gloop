# Gloop

Add some structure to your OpenGL code!
Object oriented OpenGL library for Crystal.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     gloop:
       gitlab: arctic-fox/gloop
   ```

2. Run `shards install`
